# Credits and Attribution

Document all third-party custom content, AI-generated source assets, licenses, and permissions here before any public release.

## How to use this file

For each third-party or AI-assisted asset:

1. Record the asset name and what it is (model, texture, script, 2DA row, etc.).
2. Record the source (Neverwinter Vault URL, forum post, GitHub repo, etc.).
3. Record the license or permission granted (NWN Vault community use, CC-BY, custom, etc.).
4. Record the imported path in this repository.
5. Note any modifications made.

Keep AI-generated mesh sources separated from hand-authored work.
