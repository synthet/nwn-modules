# Release Notes

Each release should record:

- exact NWN:EE storefront and build used for validation;
- Nasher, neverwinter.nim, NWNT, Blender, and NeverBlender versions;
- packaged `.mod`, `.hak`, and `.tlk` artifact names;
- required installation order for haks and tlks;
- third-party asset credits and license notes;
- known compatibility issues.
